Font-formating

HTML Formatting is a process of formatting text for better look and feel. HTML provides us ability to format text without using CSS
There are many formatting tags in HTML.
These tags are used to make text bold, italicized, or underlined. There are almost 14 options available that how text appears in HTML and XHTML.

Verdana (sans-serif)

Tahoma (sans-serif)
Trebuchet MS (sans-serif)
Times New Roman (serif)
Georgia (serif)
Garamond (serif)
Courier New (monospace)
Brush Script MT (cursive)
